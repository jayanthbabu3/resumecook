# Template Fix Prompt Reference

## Quick Prompt Template

Use this prompt to fix any template. Just replace `[TEMPLATE-NAME]` with the template you want to fix:

```
Check the template "[TEMPLATE-NAME]" and fix all issues found by the verification script. Follow the AUTOMATED_TEMPLATE_FIX_WORKFLOW.md to ensure all requirements are met.
```

## Examples

### Example 1: Fix corporate-blue
```
Check the template "corporate-blue" and fix all issues found by the verification script. Follow the AUTOMATED_TEMPLATE_FIX_WORKFLOW.md to ensure all requirements are met.
```

### Example 2: Fix minimalist-pro

Check the template "minimalist-pro" and fix all issues found by the verification script. Follow the AUTOMATED_TEMPLATE_FIX_WORKFLOW.md to ensure all requirements are met.
```

### Example 3: Fix classic-elegant

Check the template "classic-elegant" and fix all issues found by the verification script. Follow the AUTOMATED_TEMPLATE_FIX_WORKFLOW.md to ensure all requirements are met.
```

## Shorter Version (Alternative)

If you prefer a shorter prompt:

```
Fix the template "[TEMPLATE-NAME]" - run verification and fix all issues.
```

## How to Use

1. Copy the prompt template above
2. Replace `[TEMPLATE-NAME]` with your template name (e.g., `corporate-blue`, `minimalist-pro`, etc.)
3. Paste it in the chat

## Template Name Format

- Use **kebab-case** (lowercase with hyphens)
- Examples: `corporate-blue`, `minimalist-pro`, `classic-elegant`, `business-modern`

## What Happens

When you use this prompt, the system will:
1. Run the verification script on the template
2. Identify all issues
3. Fix them following the workflow
4. Verify the fixes
